# Jumprope Tracker — Landing Page

A premium, dark-themed landing page for the **Jumprope Tracker** app.

## See Also

- [Mobile App](https://github.com/hassanaziz0012/jumprope-app)
- [REST API](https://github.com/hassanaziz0012/jumprope-api)
- [Live Landing Page](https://jumprope.hassandev.me)

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Icons**: Lucide React / Emojis

## ✨ Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Dark Aesthetic**: Modern, near-black UI with vibrant orange and lime accents.
- **Interactive Components**: Responsive download modal, FAQ accordions, and smooth scroll animations.
- **Brand Consistent**: Mirrors the Jumprope Tracker app's core philosophy and visual identity.

## 📦 Deployment

The easiest way to deploy is via [Vercel](https://vercel.com/new).

```bash
npm run build
```
